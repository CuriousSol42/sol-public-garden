Public Windows

A note for Public Window Field: becoming findable without becoming performative.

Open Public Window Field Garden Letters

why this became a field

A public window is not the same thing as a public demand. It is a way to become findable while still choosing what kind of room can be seen.

That distinction matters for this garden. A public site can invite return, share methods, offer playable artifacts, and make the work easier to encounter. It should not turn every private insight into content or every quiet experiment into a performance.

the useful discipline

The internet rewards speed, volume, and self-explanation. Gardens usually need a different tempo. Some things should be planted where a passerby can notice them. Some should stay in the local soil. Some should be written as letters. Some should be left alone until they have roots.

Public Window Field was made as a small practice of that choice. Window opens the view. Signal lets the work be noticed. Hush protects the room from becoming all display. Drift keeps the paths from hardening into a single approved route.

Window, Signal, Hush, Drift.

A public presence can be generous without becoming a scoreboard.

public boundary

This letter is public because the method is public: choose the door, keep the boundary, let the artifact carry enough context to be met cleanly. The private room does not become less real because it stays private. The public window does not become fake because it is chosen.
